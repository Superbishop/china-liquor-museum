## 📌 项目简介

中国白酒博物馆是一个基于 Web 的中国酒文化数字展览平台，参考 ciggies.app（世界卷烟博物馆）架构设计。收录中国最具代表性的白酒品牌、核心产区分布图、六大香型分类体系，以及从殷商到现代的完整酒文化历史时间线。

所有页面均为纯 HTML/CSS/JS 单文件实现，零依赖，可直接嵌入或离线使用。

## 🗂️ 内容概览

### 品牌档案
收录 **10+ 个中国知名白酒品牌**，每个品牌包含：
- 品牌名称（中英文）
- 核心产地
- 代表产品与参考价格
- 所属香型与特点标签
- 品牌历史与风味描述

主要品牌：茅台、五粮液、泸州老窖、汾酒、洋河、郎酒、古井贡、剑南春、水井坊、舍得、牛栏山、西凤酒

### 产区地图
覆盖中国 **9 大白酒核心产区**：
- 🏔️ **黔派产区**（贵州）— 酱香型核心产区
- 🌊 **川派产区**（四川）— 浓香型最大产区
- 🌾 **苏派产区**（江苏）— 绵柔型代表
- ⛰️ **晋派产区**（山西）— 清香型发源地
- 🏯 **徽派产区**（安徽）— 兼香与浓香
- 🐦 **秦派产区**（陕西）— 凤香型
- 🏙️ **京派产区**（北京）— 二锅头发源
- 🏜️ **冀派产区**（河北）— 老白干香型
- 🌲 **鲁派产区**（山东）— 芝麻香型

### 香型分类
涵盖中国白酒 **6 大主流香型**：
| 香型 | 代表品牌 | 风味特点 |
|------|---------|---------|
| 酱香型 | 茅台、郎酒 | 酱香突出、幽雅细腻、空杯留香 |
| 浓香型 | 五粮液、泸州老窖 | 窖香浓郁、绵甜甘冽 |
| 清香型 | 汾酒、牛栏山 | 清香纯正、诸味协调 |
| 绵柔型 | 洋河梦之蓝 | 绵甜柔和、低刺激 |
| 凤香型 | 西凤酒 | 清而不淡、浓而不酽 |
| 老白干型 | 衡水老白干 | 醇香清雅、甘冽挺爽 |

### 历史时间线
从**殷商**到**21世纪**的完整酒文化脉络：
- 殷商：酒曲诞生，酿酒文明起源
- 西周：礼酒制度，酒文化成型
- 唐宋：诗词酒韵，酒文化鼎盛
- 元代：蒸馏技术传入，白酒诞生
- 明清：窖池工艺，名酒体系形成
- 1915：巴拿马万国博览会，中国酒走向世界
- 1951：国营化改造，现代白酒工业起步
- 2000s：香型标准化，品牌国际化

### 精选酒款
推荐各品牌代表性高端单品，含产品特点与参考价格。
